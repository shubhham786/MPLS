The program simulates the behaviour of a queueing system under given cases.

Compilation and execution environment: Linux g++

Steps to run the program
1. Compile the program using command: g++ Answer2.cpp 
2. Run using ./a.out
3. Choose a case to simulate from the provided 3 different cases.
4. Input the values for lambda, mu and the time for which you want to run the simulation.
5. Outputs are displayed on the console as well as saved in a .txt file with appropriate names.
